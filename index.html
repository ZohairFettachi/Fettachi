Mes derniers travaux

Site Web E-commerce Full Stack

Description

Ce site e-commerce full-stack permet aux utilisateurs de parcourir les produits, de les filtrer et de les trier par catégorie ou prix, puis d’ajouter des articles à leur panier en sélectionnant des variantes comme la taille. Les utilisateurs peuvent créer un compte, se connecter et passer une commande en renseignant une adresse de livraison et en choisissant un mode de paiement : paiement à la livraison ou en ligne via Stripe et Razorpay.

Le panneau d’administration offre une gestion complète de la boutique. Les administrateurs peuvent ajouter, modifier et supprimer des produits, consulter et gérer les commandes des clients, ainsi qu’actualiser leur statut. Cela permet d’assurer une gestion efficace du magasin et des transactions.

Technologies utilisées

  • React.js: Utilisé pour construire l'interface dynamique et réactive, avec une architecture basée sur les composants.
  • Tailwind CSS: Utilisé pour le style de l'application avec une approche de CSS utilitaire, permettant un design réactif et personnalisable.
  • React Toastify: Permet d'afficher des notifications (toast) pour informer l'utilisateur des actions réalisées.
  • Node.js: Utilisé pour l'intégration backend et la gestion de la logique côté serveur.
  • Express.js: Framework pour la création d'API backend et la gestion des routes.
  • MongoDB: Base de données NoSQL pour stocker les données utilisateurs, produits et commandes.
  • Stripe: Intégration du système de paiement en ligne.
  • JWT (JSON Web Tokens): Pour l'authentification des utilisateurs et la gestion des sessions.
  • Bcrypt.js: Utilisé pour le hachage des mots de passe afin de garantir une authentification sécurisée.
  • Vercel: Utilisé pour le déploiement du frontend de l'application.

Site Web E-commerce

FULL STACK E-commerce Website

Description

Ce projet consiste à créer une application React permettant aux utilisateurs de consulter les dernières nouvelles du monde entier, couvrant des catégories comme la technologie, les affaires, la santé, la science et le sport. L'application récupère les informations via l'API NewsAPI et les affiche dans une interface conviviale. Elle inclut un navbar interactif, des cartes d'articles avec des images et des descriptions, ainsi que la possibilité de filtrer les nouvelles par catégories. Les utilisateurs peuvent choisir parmi ces catégories pour accéder aux articles les plus récents et pertinents provenant de sources mondiales fiables.

Technologies utilisées

  • React: Bibliothèque JavaScript pour créer l'interface utilisateur dynamique et réactive.
  • NewsAPI: API utilisée pour récupérer les dernières actualités de sources mondiales fiables.
  • CSS: Langage de style pour personnaliser l'apparence et la mise en page de l'application.
  • Bootstrap: Framework CSS utilisé pour créer des interfaces réactives et modernes, avec des composants prêts à l'emploi pour une meilleure mise en page.

Porjet Frontend

News

News

Description

L'application récupère les données météo actuelles depuis l'API OpenWeatherMap et affiche des informations comme la température, les conditions météorologiques, l'humidité et la vitesse du vent en fonction de la ville saisie. Une barre de recherche permet à l'utilisateur d'entrer le nom de la ville et d'obtenir les informations météo correspondantes.

Technologies utilisées

  • HTML: Utilisé pour la structure de la page web et du formulaire de recherche.
  • CSS: Utilisé pour le design et la mise en page de l'application, y compris l'affichage des informations météo.
  • JavaScript: Utilisé pour la logique de l'application, comme la gestion des requêtes API et l'affichage dynamique des données météo.
  • OpenWeatherMap API: Utilisé pour récupérer les données météorologiques en temps réel selon la ville saisie par l'utilisateur.

Projet Frontend

Application météo

Liste de tâches

Description

Voici un projet pour créer une To-Do List avec HTML, CSS et JavaScript. Vous pourrez ajouter, marquer et supprimer des tâches, tout en utilisant le stockage local pour sauvegarder votre liste même après la fermeture du navigateur.

Technologies utilisées

  • HTML: Utilisé pour la structure de la page web.
  • CSS: Utilisé pour le design et la mise en page de l'application.
  • JavaScript: Utilisé pour la logique de l'application, comme l'ajout, la suppression et la mise à jour des tâches.
  • Local Storage: Utilisé pour sauvegarder les tâches localement dans le navigateur, même après sa fermeture.
  • DOM (Document Object Model): Utilisé pour manipuler les éléments HTML (ajouter, supprimer, ou marquer des tâches).
  • Event Listeners: Utilisés pour détecter les interactions de l'utilisateur, comme cliquer sur des tâches ou en ajouter de nouvelles.
